Code iterates through messages in the inbox that do not have **FamilyExpenses** label and process the ones that have **Kupiłeś i zapłaciłeś:** in the subject.

For each message it extract:

- Date	
- Id	
- FullPrice	
- Item	
- All Items cost	
- Multiplication	
- Item count	
- Item price

After putting it into spreedsheet, adds the **FamilyExpenses** label.
